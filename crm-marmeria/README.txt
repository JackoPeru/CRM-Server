===========================================
           CRM MARMERIA v1.0.0
===========================================

Sistema CRM professionale per la gestione di marmerie e lavorazioni in pietra.

===========================================
REQUISITI DI SISTEMA
===========================================

- Windows 7 o superiore (64-bit)
- 4 GB di RAM (consigliati 8 GB)
- 500 MB di spazio libero su disco
- Connessione Internet (per alcune funzionalità)

===========================================
CARATTERISTICHE PRINCIPALI
===========================================

✓ Gestione Clienti e Fornitori
✓ Catalogo Materiali e Prezzi
✓ Gestione Progetti e Preventivi
✓ Monitoraggio Ordini
✓ Dashboard Analytics
✓ Backup e Sincronizzazione Dati
✓ Interfaccia Moderna e Intuitiva

===========================================
PRIMO AVVIO
===========================================

1. Avvia l'applicazione dal menu Start o dal desktop
2. Il server interno si avvierà automaticamente
3. L'interfaccia si aprirà nel browser integrato
4. Segui la procedura guidata di configurazione iniziale

===========================================
CONDIVISIONE DATI
===========================================

Per condividere i dati con altri PC:
1. Vai in Impostazioni → Condivisione Dati
2. Avvia il server di condivisione
3. Condividi l'indirizzo IP mostrato con altri utenti
4. Altri PC potranno accedere tramite browser web
